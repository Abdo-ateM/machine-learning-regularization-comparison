# Project Summary

This project compares L1, L2, Dropout, Batch Normalization, and Elastic Net regularization techniques on MNIST using TensorFlow/Keras.

The included paper explains the motivation, methodology, and conclusions, while the notebook and scripts provide the implementation.

## Best reported result

Batch Normalization achieved the highest reported test accuracy: **98.27%**.
